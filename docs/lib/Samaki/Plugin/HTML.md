NAME
====

Samaki::Plugin::HTML -- Display HTML content in browser

DESCRIPTION
===========

Write cell content to an HTML file and open it in the default browser.

OPTIONS
=======

No specific options.

EXAMPLE
=======

    -- html:page.html
    <html>
      <body>
        <h1>Hello, World!</h1>
        <p>This is HTML content.</p>
      </body>
    </html>

Output: Creates `page.html` and opens it in your default browser.
