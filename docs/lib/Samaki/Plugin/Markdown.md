NAME
====

Samaki::Plugin::Markdown -- Render Markdown as HTML

DESCRIPTION
===========

Convert Markdown content to HTML using [Markdown::Grammar](Markdown::Grammar) and open it in the default browser.

OPTIONS
=======

No specific options.

EXAMPLE
=======

    -- markdown:doc.html
    # Hello, World!

    This is **Markdown** content with:

    * Lists
    * Links
    * _Emphasis_

Output: Creates `doc.html` with rendered HTML and opens it in your default browser.
